# Cluster Visualization Images

This directory contains the clustering visualization outputs generated by `dataset-generation-part-1.ipynb`.

## Expected Files

Place your cluster visualization images here with the following naming convention:

-   `hotel_clusters.png` - Hotel domain 3-panel visualization
-   `banking_clusters.png` - Banking domain 3-panel visualization
-   `flight_clusters.png` - Flight domain 3-panel visualization
-   `insurance_clusters.png` - Insurance domain 3-panel visualization
-   `retail_clusters.png` - Retail domain 3-panel visualization
-   `telecom_clusters.png` - Telecom domain 3-panel visualization

## Format

Each image should contain the 3-panel output:

1. **Left Panel**: Elbow Method (Inertia) graph
2. **Middle Panel**: Silhouette Analysis graph with optimal k marked
3. **Right Panel**: FPS Strategy Map with PCA projection and selected samples

These are automatically generated when you run the pipeline and can be saved from the matplotlib output.
